Anomaly-Aware Flood Expansion Detection and Prediction is a deep learning-based framework designed to improve flood monitoring and forecasting by integrating Convolutional Neural Networks (CNN) 
and Long Short-Term Memory (LSTM) networks. The system detects flood-affected regions from satellite imagery and predicts future flood expansion using historical rainfall and water-level data.

To improve reliability, an anomaly detection module identifies and filters noisy or inconsistent data caused by sensor errors or unexpected environmental changes. 
This enhances prediction accuracy, reduces false alarms, and supports effective disaster management.
